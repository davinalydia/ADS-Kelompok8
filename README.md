IPB Campus Facility & Queue System merupakan proyek kelompok pada mata kuliah **Analisis dan Desain Sistem** yang dikembangkan secara bertahap mulai dari analisis kebutuhan, perancangan sistem berorientasi objek, hingga implementasi dan pengujian aplikasi.

Sistem ini dirancang untuk mengelola peminjaman fasilitas kampus dan sistem antrian secara terstruktur. Mahasiswa dan dosen dapat melakukan pemesanan fasilitas, masuk ke dalam antrian, serta melihat riwayat peminjaman. Admin berperan dalam mengelola data fasilitas, memvalidasi peminjaman, dan memantau penggunaan fasilitas.

Proyek ini mengikuti alur pertemuan praktikum yang mencakup **analisis, desain, arsitektur, implementasi, hingga pengujian dan presentasi akhir**.
